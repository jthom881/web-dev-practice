## 1. Project Overview

This project is a responsive multi-page website built using HTML and CSS. The goal of this project is to demonstrate the principles of responsive web design, including mobile-first development, flexible layouts, and consistent styling across multiple pages.

The website was deployed using GitHub Pages and is accessible on desktop, tablet, and mobile devices.

---

## 2. Design Explanation

This website follows modern responsive web design best practices:

### Mobile-First Design
The website is designed to function on mobile devices first, then expand for larger screens using CSS media queries.

### Media Queries
Media queries were used to adjust layout and navigation based on screen size. This ensures usability on mobile, tablet, and desktop devices.

### Flexbox Layout
Flexbox was used to create a flexible grid system for layout structure and responsive card design.

### Consistent Styling
A single external CSS file is used across all pages to ensure consistency in:
- Colors
- Fonts
- Spacing
- Layout structure

### Box Model Usage
Proper use of margin, padding, and borders improves spacing, readability, and layout structure.

## 3. CSS Implementation

The CSS file includes:
- Navigation bar styling
- Hover effects for links
- Responsive grid layout using Flexbox
- Card-based content sections
- Mobile-friendly layout adjustments

A consistent dark navigation bar and light content background were used for visual contrast and readability.

## 4. Testing Report

- Result: Full layout displayed correctly with horizontal navigation and multi-column layout.

### Tablet Testing
- Environment: Browser resized to tablet width
- Result: Layout adjusted spacing and stacked content appropriately.

### Mobile Testing

- Result:

  - Content displayed in a single column
